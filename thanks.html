We Want to Thank Some People

We all had to get our start some where, get help from others, have problems debugged, code explained, and surely a lot more than just that. I, Joe Maples, appreciate that, and I'm sure every other developer out there does. I'd like to say thank you to some people, and I'm sure anyone else would say the same. Here's to all the great devs out there.
Team Vanir, and more specifically PrimeDirective and NuclearMistake, for helping me build my first ROM and continuously putting up with my anyoing questions when I was 13.
JustArchi, for help setting up his great optimizations, but more importantly, creating them in the first place.
Matt Bloomfield, for help with silly mistakes and errors
MrRobinson, for helping me
